Haxe Roundup № 309

by Skial Bainn published on Sunday 1st March 2015
Suggest a link Correct this issue

Slava Tretyak has written Fast Iterators for StringMap in which he benchmarks the StringMap Iterator implementation from Haxe 3.1.3, the Haxe GitHub repo and his own iterator implementation, all compiled to JavaScript running on Chrome, Safari and Firefox.

Sergio Barrio has written his final article for his HaxeFlixel tutorial series on creating your first 2D Shoot'em up. The entire tutorial source code is available on GitHub. Another great contribution to the community.

Adrien Fischer has posted a sneak peek of a tutorial he is working on, called Coloring, a puzzle board game where you merge cards to unlock further colours.

Lubos Lenco has ported Flump, which “exports Flash .fla's to GPU friendly formats”, from Flambe to Kha available from the aptly named flump_kha.

Lubos also has released a new demo, I assume using his ZBlend library, which shows Blender baked textures in Kha.

The Shiro Games team have made Evoland available for Linux through Steam. If you enjoyed Evoland you should probably keep an eye on Evoland 2, I really like the quality art work on the dev blog.

And to finish this roundup off is FireCHIP-8 by Ohminivore. FireCHIP-8 is a CHIP-8 emulator which abstracts the rendering engine but provides a default OpenFL implementation. You can checkout the demo which runs Pong.
