# Haxe Roundup № 159

Flambe 3.1.0 "is hot off the presses!". Checkout the 3.1.0 announcement for all the new goodness.

Tong has updated the Haxe / Haxelib bash auto completion scripts for all the extra skilled people out there.

Lubos has released his new game Pexeso, the "beautiful memory game" to the Google Play store with plans to add "cross-platform online multiplayer using [Nuggeta][link 9] and [OpenFL][link 10]”.

A new minor release of OpenFL is on haxelib "with 64-bit Neko support for Mac and minor improvements to keep everything running smoothly".

Dennis has written about using gestures in OpenFL with the roxlib library which isn't dependent on any native features.

Playtomic the free, open source, self hosted tools for game developers have announced that they are working on a Haxe API.

German Allemand has released a new signalling library, HxSignal which uses operator overloading through abstract types, allowing for an interesting coding style.
