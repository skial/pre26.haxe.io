Updates from the Haxe core

35 commits have been made to the Haxe compiler in the last week.

  • Allow using Const type parameters as value.
  • Make sure multiple @:build macros are run.
  • Add HXCPP_OPTIMIZE_FOR_SIZE pull request.

Congrat’s to Sven for fostering such a dedicated community within the last year, it is well deserved.

Franco’s doom library is particularly interesting, providing just the DOM View API, which was announced on the Haxe mailing list, ready for other libraries to build on.
