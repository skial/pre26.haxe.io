# Haxe Roundup № 140

HaxePunk seems to be getting faster according its BunnyMark port, with 5000 bunnies at 30+ fps. All based on the latest dev branch.

And it seems both the lead devs of HaxeFlixel and HaxePunk, Alexander Hohlov and Matt Tuttle, respectively, might be planning on combining efforts to bring more speed to both projects.

The in progress Objective-C target has successful built the NME PiratePig demo. Head over to the mailing list for a better description and a picture!

Zest3D, the cross platform 3D engine for Ouya, iOS, Android, BlackBerry, PC, Mac and more *"will be coming to Haxe soon"*.

Lars Doucet posted over at HaxeFlixel's forum how he has *"Visual C++ debugging the CPP target"* working, and for the adventurous people, the project files.

How about this for a discussion, Andreas Rønning talks about how he is "baffled and frankly a little disappointed in the lack of support for Haxe NME among Flash Developers", their reaction to Loom, "issues with Unity3D's 2D performance" and more. Take some time to read all of the 69 70 comments.

This looks crazy, NME WorldBuilder the "ultimate tool for making crossplatform games" and with a huge feature list. While NME looks down, checkout the youtube video for 14 minutes of Nape goodness.

Theres one more NME UI framework to add to the list, Zajac which provides the "ability for styling over CSS", supported targets are CPP, Neko and Flash and its been tested on iOS, Android, BlackBerry, Windows and Flash.

StablexUI, one of the first? NME UI frameworks, is now able to create loops in xml. Also a new sample game has been released, Ladybird, where all the objects are StablexUI widgets.

This is bound to be useful, protohx a "crossplatform implementation of Google's Protocol Buffers" has been released on haxelib.

With Haxe 3 possibly being released at the end of April, the future of haxelib has come up, so it's a good time to put your ideas forward. Jason O'Neil has received the Haxe Foundations approval to take charge.There's also a road map for the new Haxelib.

This looks awesome, planetize.me, the 3D space simulation game, using Haxe PHP and Flare3D.

Thanks to those who emailed me updates, articles etc. More please!
