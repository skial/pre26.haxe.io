# Haxe Roundup № 122

A new release of FlashDevelop is out with improved NME and Haxe AIR support and numerous CSS, HTML5 and Javascript improvements.

Pah Arif has fixed an HXCPP Android bug on Linux with all the details on his blog.

Steeven Plu created Milady in 48 hours, LudumDare Mode.

Another game Make a Scene: Dinosaurs is out now for the HP touchpad, made with NME.

Continuing with games, two interesting competitions are running this month, Github Game Off and TIGSource's TIGCompo. Adrian over at Vigeo Games shows a prototype for TIGCompo. Looks brilliant!

It looks like Nicolas is working on a lightweight stage3d engine for Haxe, which will have a 2D framework on top of it soon.

This is pretty cool, a 3D renderer ENTIRELY in CSS3. Check out the demo, your likely need the latest Chrome or Firefox to view it.

DynaLight currently only Haxe/Flash is a Dynamic Lighting engine. The source is available from github. I forgot to put this in the last roundup.

Slightly going off the Haxe topic, but with WebRTC now live in Chrome v23 some pretty interesting demo's / apps can be created. Oh, Tongs hx.html5 library has WebRTC type definitions.

And lastly, for everyone working on touch devices, Responsive Navigation : Optimizing for Touch Devices is definitely worth reading.
