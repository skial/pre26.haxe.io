# Haxe Roundup № 139

If you've checked out Haxe 3's Abstract Types, take a look at this short and simple example by the macro master Juraj Kirchheim.

Andy Li has written a great article called "HXML tricks every Haxe user should know".

The Haxe Flixel tutorials keep coming, this time showing off the "Mouse Interaction Manager". Touch events should work but they are currently untested.

Alexander Hohlov tweeted that Haxe Flixel's dev branch has "initial support for texture atlas loading" thanks to Sergey Miryanov.

Jaroslav Meloun has ported Senocalur's TransformTool (Github: jarnik / transformtool, License: unknown) to Haxe. Currently tested Flash and Neko targets.

Tong has once again had a late night

  • porting nekoboot to Haxe.

    • François Nicaise continues to share his experience from making Strange Masks, this time for the "NME folks", called "Speed".

    Grunt Haxe (Github: Fintan / grunt-haxe, License: MIT) now works with GruntJS v0.4.

    This is brilliant news, it looks like BlackBerry is going to contribute to NME.

    Suat Eyrice has created and open sourced NME Webview (Github: SuatEyrice / NMEWebview, License: unknown) for NME's iOS and Android targets.

    The IntelliJIDEA Haxe plugin has just been updated with code folding.
