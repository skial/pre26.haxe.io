# Haxe Roundup № 135

NME 3.5.5 has been released, more of a maintenance release than anything, but with Native and WebGL GLSL shader examples.

Two new UI libraries that all the NME folks should checkout :

YAHUI is “a skinnable NME user interface designed to be used out the box and give a native user experience”.

StablexUI is a “macro driven UI engine for NME. All platforms are supported”.

And one more for all the NME people, you can now add / edit tutorials and docs at nme.io!

Evoland, the follow on from Evoland Classic has a write-up on VentureBeat.

Another Haxe game that has received great publicity is Strange Mask, which has been reviewed by a well known French Magazine - Canard PC.

Jordan has a new release of TJSON, the tolerant json parser.

Jon Borgonia is presenting at the OUDL about Lua and Haxe on February 18th.

Flambe 2.2 has been released, fixing important Flump bugs, and now comes with an experimental WebGL renderer!

This is really awesome, Tong has a working Native Client build toolchain for HXCPP. You can get it from github.

This is something that everyone should keep an eye on, Basis which has “the goal to provide cross platform access to native UI components”.

Is Haxe Memes a sign of Haxe’s increasing popularity?
