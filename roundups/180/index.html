# Haxe Roundup № 180

Vadim continue's to create interesting demo's, this time a HTML5 Terraria profile viewer which test's his recently added OpenFL-bitfive FileReference, combined with recent fixes to ByteArray which allow's you to port feature rich applications to HTML5.

Another from Vadim, a Haxe OpenFL Bitmap Font implementation. Kerning, align, data inlining and a few extras.

OpenFL has released a new article OpenFL meets Lime. You now have access to low level rendering, audio and input.

Christopher Kaster has released openfl-tiled v0.3.1 to haxelib which add's isometric map support.

Andy Li's hard work improving the pygments library to highlight Haxe 3 correctly has found it's way to Github, so everything finally renders correctly!

Justin Donaldson has started a Nominate/vote for your favorite Haxe library of 2013 thread over on google+. Go submit / vote for your favorite library!

HaxeFlixel are moving from their in house forum to using google groups. Go and join them!

And finally, I hope everyone has a Happy Holiday!
