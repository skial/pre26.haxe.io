# Haxe Roundup № 177

The Haxe IDE indiegogo project came to an end last week and it exceeded its goal with the help of FlashDevelop and the Haxe Foundation!

Lubos Lenco's NME / OpenFL game Castle Siege has reached 100k installs on Google Play!

Vadim has released HaxMin. It is a minifier and obfuscator for the javascript output by Haxe. This is brilliant because Reflect will continue to work!!! Checkout the Introducing: HaxMin post for alot more details.

Christopher has released openfl tiled v3 which allow's developers to use the TILED map editor.

Alexander Kuzmenko has added theme support to the dev branch of StablexUI. To get a good understanding check out his post StablexUI : skins, defaults and themes. I like the idea of themes being installable from haxelib. Nice and simple.

Elliott Stoneham has released the slide's for his talk Adventures with go.tools/ssa in which he converts golang to Haxe with his yet unreleased golang to Haxe compiler.

Check out ionDrift created by b10b games using OpenFL and awe6 targeting HTML5 2D canvas.

Double Duck have released DDGameAnalytics, a OpenFL native extension for the GameAnalytics service.

Andreas Rønning has posted about his approach to a multi-threaded game loop in OpenFL. With it he is able to animate 11000 rotated and scaled bitmaps via drawTiles which also includes the usual position updates.

And finally it looks like OpenFL will be targeting Tizen.
